# React Pizza v2

The project of an online pizza ordering store

- React 18
- TypeScript
- Redux Toolkit
- React Router v6 
- Axios  
- React Hooks 
- Prettier, ESLint
- CSS-Modules / SCSS 
- React Content Loader 
- React Pagination 
- Lodash.Debounce
- Code Splitting, React Loadable, useWhyDidYouUpdate
